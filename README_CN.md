## 简介

GPUPixel 是一个高性能、跨平台的图像和视频滤镜库，体积小巧。使用 C++11 和 OpenGL/ES 构建，提供美颜滤镜功能。支持 iOS、Android、Mac、Windows 和 Linux，兼容所有 OpenGL/ES 平台。
