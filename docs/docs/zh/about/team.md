# 团队介绍

## 关于我们

GPUPixel 是一个充满活力的开源项目，由一群热爱图像处理技术的开发者共同维护。我们致力于为开发者提供高性能、易用的图像处理解决方案。

## 核心团队

GPUPixel 项目由 [Jaaron Kot](https://github.com/jaaronkot) 创建并主导开发。核心团队成员均具备丰富的图像处理经验和跨平台开发能力。

## 技术方向

- 🎨 图像处理算法优化
- 🚀 GPU 渲染性能提升
- 📱 跨平台兼容性支持
- 🛠️ 开发者工具改进
- 📖 文档和示例完善
